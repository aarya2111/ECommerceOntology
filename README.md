# E-Commerce Product Ontology

## Description
This project presents an E-Commerce Product Ontology developed using Protégé and OWL 2.0.

### Classes
- Product
- Customer
- Order
- Category
- Seller
- Payment
- Review
- ShoppingCart

### Object Properties
- placesOrder
- containsProduct
- belongsToCategory
- soldBy
- makesPayment
- writesReview
- reviewsProduct
- hasCart
- cartContains

### Data Properties
- customerName
- productName
- price
- quantity
- orderDate
- paymentAmount
- rating
